# cosmic-toolkit
宇宙工具箱


## 工具箱里有什么？
```bash
### 🐍 Python: 星球体重计算器
*   **文件:** `python/planet_weight.py`
*   **功能:** 输入你在地球上的体重，它会告诉你到了其他星球上会变成多少公斤
*   **亮点:** 每个星球都附带外星小怪的科学（瞎扯）点评

# 在你的电脑上运行
python python/planet_weight.py

💛 JavaScript: 心情戒指
文件: javascript/mood_ring.html

功能: 一个基于宇宙背景辐射（其实是随机数）的心情探测器

亮点: 会生成一个颜色和一句外星小怪风格的今日运势

在线体验: 戳这里查看心情戒指

🐚 Shell: Git 懒人脚本
文件: shell_scripts/git_lazy.sh

功能: 一键执行 git add .， git commit -m "信息"， git push。

亮点: 为了把偷懒进行到底而写
# 使用方式
bash shell_scripts/git_lazy.sh "你的提交信息"
